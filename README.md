## 题况说明

2025 年工大的计网期末 **题量非常大**，笔者以及身边同学基本都是 **卡点做完**。

题型与往年一致，包括：

- 填空（10 道）
- 判断（5 道）
- 选择（20 道）
- **最后五道大题（分值很高）**

**一定要去看当年的 408 考试原题！**  
2025 年计网的题目与当年 408 的计网原题高度重合，  
从选择题到最后一题的大题，都能在 408 中找到对应内容。

---

## 复习说明

### ✔ 如果是“长期好好上课的同学”
推荐复习方式：

- 认真看老师 PPT（覆盖面广、重点明确）
- 刷《王道计算机网络》

这两者足够覆盖大部分考点。

---

### ✔ 如果是“最后两周突击的同学”（默认从零开始）

建议节奏如下：

1. **第一周：完整看一遍王道课**  
   帮助建立完整知识框架。

2. **第二周：配合问天讲师团 PPT 强化复习**  
   - 逻辑清晰  
   - 用语准确  
   - 包含大量 408 原题  
   -（笔者已在“笔记版”中标注了少量纰漏）

3. **再用一天梳理各层协议的关系**  
   搞清楚“谁服务于谁”。

4. **最后两天**  
   - 刷往年期末题  
   - **刷当年 408 原题（极其重要）**

## 具体考题回忆

以下内容为考后回忆，仅供参考。

---

### 📝 填空题

1. OSI 模型有几层？自下而上分别是什么？
2. HTTP 如何解决无状态问题？
3. 曼彻斯特编码
4. 电路交换、报文相关定义
5. MTU 计算（除去报文头再除 8）
6. CRC 冗余校验

---

### 🧩 选择题

1. 分组交换、电路交换与报文交换的用时比较
2. FSK、PSK 等调制方式中谁的利用率最低
3. Nyquist 计算信道容量
4. 汉明距离相关计算
5. DHCP 动态请求过程的源 IP 与目的 IP
6. SMTP 与 POP3 的具体作用（发送/接收）

---

### 📘 大题

1. 在 C/S 与 P2P 架构下，分别计算文件传输总用时，并举例典型 P2P 应用
2. HTTP 协议下传输 JPG 文件的过程与总用时计算
3. 子网划分，子网聚合（CIDR）
4. CSMA/CD、二进制退避算法、传输时间计算
5. （与 408 原题高度相似的大题）
